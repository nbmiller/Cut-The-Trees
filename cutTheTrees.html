Cut The Trees - A Game for the Industrialists
This Game Is Meant To Be Played With Chrome!

The environment will magically improve greatly and global warming is no more.. Yay! (Disclaimer: The science for this does not currently exist IRL.)

Accept Defeat, but annoy environmentalists anyway (Green means icon pressed is correct, red means you still need to press it, no change means you pressed the wrong one silly!)
(Can Be Pressed More Than Once)

Enter a Size (Integer) and Press "Play Game" to randomly generate a new puzzle of the corresponding size. Don't Get Greedy.

Instructions: Match the rows with the columns! A number then a break means there is the corresponding trees in a row that must be cut and a gap of unknown size before the next number! (assuming there is another number)
